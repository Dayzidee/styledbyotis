# StyledByOtis 🧥

A premium, urban editorial online retail shop built with **React**, **Vite**, and **Tailwind CSS v4**.

## ✨ Features
- **Editorial Layouts**: Custom-designed screens following the "Styled" brand identity.
- **Priority Pre-orders**: A front-row seating system for upcoming product drops.
- **High-Fidelity Product View**: Split-layout detail pages with interactive galleries.
- **Global Bridge Tracking**: Real-time logistics visualization.
- **Mobile Responsive**: Fully optimized for small screens with a glassmorphic bottom navigation.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Dayzidee/styledbyotis.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the development server:
```bash
npm run dev
```

### Build
Generate the production bundle:
```bash
npm run build
```

## 🛠️ Tech Stack
- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS v4
- **Icons**: Material Symbols & Lucide React
- **Routing**: React Router v6
